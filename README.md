# STOCK PREDICTOR
  A Web app to predict IBM stock price
  
# USAGE INSTRUCTIONS
  * Free and open-source
   
# Features
  * Attractive UI
  * Fast and quick results
  * Valid data from Alpha vantage
  
# Built with:
  Html, css, javascript, jquery, python , flask , sklearn, bootstrap
